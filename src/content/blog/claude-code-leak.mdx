2026년 3월 31일, Claude Code의 소스코드가 통째로 유출됐다

나는 그 소스를 직접 보지 못했다

하지만 어느 개발자 분께서 깊게 [분석한 글](https://kuber.studio/blog/AI/Claude-Code%27s-Entire-Source-Code-Got-Leaked-via-a-Sourcemap-in-npm,-Let%27s-Talk-About-it?source=post_page-----96b646700059---------------------------------------)을 접했다

정말 재미있는 내용이 많이 있다

한번 읽어보면 좋을 것 같다

:::

그런데 어쩌다가 클로드 코드의 소스코드가 유출이 되었을까

악의적인 공격이 아니었다

npm에 배포된 Claude Code 패키지에 source map이 포함됐고  
그 결과 TypeScript 원본 상당량이 외부에 노출되었다  
직접적인 원인은 배포 과정에서 source map 제외가 제대로 적용되지 않았기 때문으로 보인다  
사소해 보이는 패키징 설정 누락 하나 때문에 Anthropic의 핵심이 유출되었다

:::

이전까지 AI는 막연히 두려움의 대상이었다

지금 내가 뭘 하고 살아야 할까… 내가 살아남을 수 있을까?..

그런데 이 사건이 뭔가를 보여줬다

AI에게는 사람이 필요하다

AI의 일을 대신 하려는 사람 말고, AI와 협력하는 사람이

그리고 소스 코드 안에는 재밌는 내용이 많이 있었다

**Buddy**

다마고치처럼 생긴 컴패니언 펫 시스템이다

AI에게는 사실 필요 없는 기능이다

하지만 사람이 좋아할 것을 알았기 때문에 만들었다

**KAIROS**

사용자가 돌아왔을 때 더 잘 도울 수 있도록

모두 사람을 향한 기능이다

사람 중심의 사고가 필요하지 않을까?

막연하게 떠올려본다

:::
